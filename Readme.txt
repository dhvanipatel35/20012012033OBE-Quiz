How to run the OBE-Quiz Project
1. Download the zip file
2. Extract the file and copy cp3 folder
3. open the Visual Studion (for Visual Studio)
4. Open Djengo (http://localhost/127.0.0.1:8000/)
5. Create a database with name Djengo
6. Run the Project
